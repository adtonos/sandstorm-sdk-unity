Interface SandstormVastUrlBuilder

Builder interface for additional params to Sandstorm link for the vast response. Builder can be obtained from ATSandstormSDK.CreateBuilder As a product of SandstormVastUrlBuilder.Build method an url is provided.

SetAdTonosKey(String)

Sets AdTonosKey. It must be provided to let SandstormVastUrlBuilder build correct url. If AdTonosKey is not provided then exception will be thrown during [SandstormVastUrlBuilder.build] Where to find AdTonosKey? It's provided by AdTonos on the portal and can be extracted from the link https://play.adtonos.com/xml/XXXXX/vast.xml, where XXXXX is the AdTonos key. AdTonos usually provides two links, one for testing purposes and one for release. During development testing key shall be used.
