Namespace Sandstorm

Classes

ATSandstormSDK

Shared object for VAST advertisements handling. This is the place where developer should start reading. ATSandstormSDK is an object containing a set of methods for generating and playing VAST advertisements. See Quick Guide for more information.

Logs

Helper class for logging interface

SandstormAd

Internal object describing an Ad.

SandstormError

Contains information about error that happened during ad requests

SandstormInvalidKeyException

Exception triggered when an invalid Sandstorm key is entered or when no key is passed to SandstormVastUrlBuilder at all. Thrown when SandstormVastUrlBuilder.Build is executed.

SandstormInvalidLanguageException

Exception triggered when an invalid language (not in ISO-639-1 or ISO-639-2 format) is entered. Thrown when SandstormVastUrlBuilder.Build is executed.

Interfaces

Sandstorm

Interface for common url and library initialization. Use ATSandstormSDK.

SandstormCallback

Interface is used to observe player events

SandstormLogger

Logger interface for the library.

SandstormVastClearable

An auxiliary interface to indicate implementations that can be cleared of advertising data when playback is complete.

SandstormVastUrlBuilder

Builder interface for additional params to Sandstorm link for the vast response. Builder can be obtained from ATSandstormSDK.CreateBuilder As a product of SandstormVastUrlBuilder.Build method an url is provided.
